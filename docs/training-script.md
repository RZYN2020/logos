3. 填写规则信息：

## 附录

### 视频制作说明

1. **录制工具**：OBS Studio 或 Camtasia
2. **分辨率**：1920x1080
3. **帧率**：30fps
4. **音频**：清晰普通话，背景无噪音
5. **字幕**：添加中文字幕

1. 上传到公司内部视频平台
