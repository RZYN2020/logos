## 快速入门

### 登录系统

1. 打开浏览器访问 `http://localhost:3000`
2. 使用默认账号登录：
   - 用户名：`admin`
   - 密码：`admin123`

**重要：首次登录后请立即修改默认密码！**

### 主界面介绍

系统主界面包含三个主要标签页：

- **规则配置** - 管理和配置日志规则
- **日志分析** - 查询和分析日志
- **系统信息** - 查看系统状态和健康检查

## 规则配置

### 创建规则

1. 点击「规则配置」标签页
2. 点击「新建规则」按钮
3. 填写规则信息：
   - **规则名称**：唯一标识符，如 `error-log-filter`
   - **规则描述**：描述规则的作用
   - **优先级**：数字越小优先级越高
   - **启用状态**：勾选表示规则生效

4. 配置条件（JSON 格式）：

```json
[
  {
    "field": "level",
    "operator": "=",
    "value": "ERROR"
  },
  {
    "field": "service",
    "operator": "in",
    "value": ["api", "web"]
  }
]
```

常用操作符：
- `=` 等于
- `!=` 不等于
- `>` 大于
- `<` 小于
- `in` 在列表中
- `contains` 包含

5. 配置动作（JSON 格式）：

```json
[
  {
    "type": "filter",
    "config": {
      "sampling": 1.0
    }
  }
]
```

动作类型：
- `filter` - 过滤/采样
- `drop` - 丢弃日志
- `transform` - 转换日志

6. 点击「保存」按钮

### 编辑规则

1. 在规则列表中找到目标规则
2. 点击「编辑」按钮
3. 修改规则配置
4. 点击「保存」按钮

规则版本号会自动递增，保留历史记录。

### 验证规则

1. 在规则表单页面，点击「验证规则」按钮
2. 系统会检查规则的语法和逻辑
3. 如果验证失败，会显示具体错误信息

### 测试规则

1. 在规则表单页面，点击「测试规则」按钮
2. 系统会使用模拟数据测试规则匹配
3. 查看测试结果，确认规则按预期工作

### 查看规则历史

1. 在规则列表中点击「历史」按钮
2. 查看该规则的所有历史版本
3. 可以查看每个版本的具体内容

### 回滚规则

1. 在规则历史页面，选择要回滚的版本
2. 点击「回滚」按钮
3. 系统会创建一个新版本，内容与选中版本相同

### 删除规则

1. 在规则列表中点击「删除」按钮
2. 确认删除操作
3. 规则会从数据库和 ETCD 中删除

### 导入/导出规则

**导出规则：**

1. 点击「导出」按钮
2. 系统会下载所有规则的 JSON 文件

**导入规则：**

1. 点击「导入」按钮
2. 选择之前导出的 JSON 文件
3. 确认导入

## 日志分析

### SQL 查询

1. 点击「日志分析」标签页
2. 在查询编辑器中输入 SQL 查询：

```sql
SELECT * FROM logs
WHERE level = 'ERROR'
  AND timestamp > NOW() - INTERVAL 1 HOUR
ORDER BY timestamp DESC
LIMIT 100;
```

3. 点击「执行查询」按钮
4. 在右侧查看查询结果

### 快捷查询

系统提供常用快捷查询：

- 🔴 **最近错误日志** - 查询最近的 ERROR 级别日志
- 📊 **按服务统计** - 统计各服务的日志数量
- 📊 **按级别统计** - 按日志级别统计
- 👥 **活跃用户排行** - 显示最活跃的用户

### 日志挖掘

日志挖掘功能可以自动发现日志中的模式：

1. 选择要分析的日志范围
2. 点击「模式挖掘」按钮
3. 查看挖掘结果：
   - 模式内容
   - 出现频率
   - 严重程度
   - 示例日志

### 异常检测

异常检测功能可以发现与基线不同的日志：

1. 选择基线日志（正常时期的日志）
2. 选择当前日志（待检测的日志）
3. 点击「异常检测」按钮
4. 查看检测结果，识别异常模式

### 智能推荐

系统可以根据日志模式推荐规则：

1. 点击「规则推荐」按钮
2. 系统分析日志模式
3. 生成推荐规则列表
4. 可以选择应用推荐规则

## 系统管理

### 查看系统信息

1. 点击「系统信息」标签页
2. 查看以下信息：
   - 系统名称
   - 版本号
   - Etcd 版本
   - 运行时间

### 健康检查

系统信息页面显示健康状态：

- **整体状态**：健康/异常
- **Etcd 连接**：已连接/未连接

### 用户管理（管理员）

**创建用户：**

1. 进入「用户管理」页面
2. 点击「新建用户」
3. 填写用户名、密码、邮箱
4. 选择用户角色

**修改密码：**

1. 点击右上角用户名
2. 选择「修改密码」
3. 输入旧密码和新密码
4. 确认修改

### 角色权限

系统支持以下角色：

- **admin** - 管理员，拥有所有权限
- **user** - 普通用户，可以查看和管理规则

## 常见问题

### 规则不生效？

1. 检查规则是否已启用
2. 检查规则条件是否正确
3. 使用「验证规则」功能检查语法
4. 使用「测试规则」功能检查匹配

### 无法连接后端？

1. 检查后端服务是否运行
2. 检查网络连接
3. 查看浏览器控制台错误信息

### 日志查询慢？

1. 缩短查询时间范围
2. 减少返回数量（使用 LIMIT）
3. 添加合适的索引

### 忘记密码？

联系系统管理员重置密码。

### 如何备份配置？

1. 使用「导出规则」功能下载规则配置
2. 定期备份数据库和 ETCD
